Navigation

The navigation bar that appears at the top of each page can be used to navigate the database and select common functions. Alternatively return to the home page at any time by clicking ‘Artemis’ at the top left of the screen.



Search

The search function can be accessed from the search bar that appears at the top of all pages or from the search button on the home page. To find a record simply enter search terms in the box. The search function works best with unique identiffiers for example part or assembly ID. It is possible however to search based on any facet.

Example

Type ‘c8m’ in the search box to return all parts whose name contains ‘c8m’

Filtering - Faceted Search

Once you have completed a search you may wish to refine your results. In order to filter your search results and exclude unwanted results you can use the ‘add facet filters’ function.

In the search view the column to the left of the search results contains a box titled ‘add facet filters’ - simply select the terms that you want to include. Once selected the relevant terms will appear at the top of left column. The relevant term in the ‘add facet filters’ box should also become highlighted.

If any of the terms you chose contain multiple options then select the small ‘+’ symbol next to the filter, this will present the various options available. By clicking on the relevant value you can limit search results to that category.

If there are a large quantity of options you can use the alphanumeric pad that will appear directly below the filter to find the term you are looking for. Additionally you can sort the results to find the term you are looking for using the functions in the bottom row of the box.

To remove a search filter simply click the ‘x’ symbol next to relevant category.

Example

To filter for items that have a specific batch ID select ‘BATCH_ID’ in the ‘add facet filters’ box. You should see that ‘BATCH_ID’ is now highlighted. At the top of the column a menu item ‘+ batch_id’ will appear. Click on the ‘+’ symbol and the available options will expand below. You can filter these options using the alphanumeric pad.

Click on the batch ID number that you are looking for

However, note that facet filtering is only valuable on particular fields. For example, if you facet on a field that is unique across all records, such as the record ID, you will get little benefit - as there will be only 1 result for every value. In this case, you would be better searching for the ID directly, or if you do not know it, searching for other bits of information you have about the record you are looking for.



Add New Part

To add a new part select the ‘New part’ option in the navigation bar that appears at the top of each page. Alternatively select ‘New Part’ button on the homepage.

You will be presented with a blank parts record including basic fields common to most parts. Where there is information available corresponding to a field enter the details. For those fields without information leave blank.

Once the fields are updated click ‘save’ to ensure changes are stored.

Default fields include:

  • type - This is the type of record either part or assembly
  • assembly - This is the ID of the assembly which this part is currently assembled
  • name - This is the text description of the part. Include any alternative names
  • drawing - This is an attachment
  • assembled_by - This is the name of the Artemis employee that assembled the assembly
  • assembled_date - This is the date that assembly was created. Please enter in the format yyyy-mm-dd hh:mm:ss
  • inspected_by - This is the name of the Artemis employee that inspected the part
  • inspected_date - This is the date that the part was inspected. Please enter in the format yyyy-mm-dd hh:mm:ss
  • assembly_history - This is a list of the assembly IDs of assemblies the part used to belong to.
  • location - This is the current location of the part.
  • location_history - This is a list of the locations the part has been.

To add a new field select the ‘+’ symbol at the bottom of the table. A new row will appear with two fields. In the left box enter the field name. In the right box enter the field value.



Add Multiple New Parts

If you wish to create multiple new parts at the same time, this can be done via the Multiple New Parts option.

This works as per the "Add New Part" option, except that you can specify you many parts to create. Note, of course, that all the new parts being created must have identical information provided for them. Once saved, you will be provided with a list of the IDs for the parts that were created.



Create a group of IDs

This is the same as adding multiple new parts - if, for example you wish to give a group of IDs to a particular manufacturer, just choose to Add Multiple New Parts. Then, on the new part form, specify the number of parts to create but provide no further information. Alternatively, add some information to identify the reason the records were created - e.g. a note such as "issued to manufacturer X" or perhaps a "manufacturer" field.

Add part to Assembly

In the Assembly record page enter the part IDs for the relevant part(s) in the box in the right hand column. Enter the IDs of parts you want to add to this assembly, separated by commas. If you do not know the ID, try searching based on other information about the part.

Alternatively, when viewing a particular part, click the "add part to assembly" button and type in the ID of the assembly to which you wish to add the current part. The page will be refreshed once you submit, and you will see the assembly information on the right hand side.
